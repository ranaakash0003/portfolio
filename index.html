These are the projects I have worked on.

Orbit

A web application where any registered user can post, search and view their lost item as well as location side by side in the google map; where a marker will be added with every post location. There is two section one for the lost post’s and one for the found post’s. To make the search result more relevant and accurate, applied keyword based search algorithm TF-IDF and for generating relevant item suggestion used machine learning where a custom kernel was used alongside with linear kernel, rbf kernel and sigmoid kernel to show the search difference among them.

GitUser-Finder

It’s a site created using React js library and GitHub api. Here user can search any GitHub user id and view the user profile, followers, following, public repositories and five latest repository name and for more details they can click to them and which redirects them to the GitHub page.

MovieStore API

Here I have created backend services for the MovieStore app. Here user can register themselves and can upload movies. There is category of movies, they can create, update, delete the genre also. Registered Customer can rent any movies from here, it’s crated using Node js and Express framework and fir the authentication JWT authentication has been used.

Bookbit

It’s book store where user can upload, search different kinds of books. User can search his desired book using the writers name and publish date. Can view the description of the book also.

Smart Hospital Finder

It's a web application, where Any hospital/diagnostic center can add their information. User both registered and unregistered may search any specific available service and result will include hospitals, doctors, prices, available hours for a certain disease or location if the matches.

Blog

It’s a simple blog where user can register and login to their profile. They can post about any topic. There is profile option where user can use their image to the profile picture. They can create, update and delete their post. The database here is used is PostgreSQL and some custom css and bootstrap are used for the design.
